Learn Java in One Day

Level 1: Variables

In Java, a variable is a container that holds data. To declare a variable, you use the following syntax:

int myNumber = 5;

Try creating a variable named myVariable and set it to the value 10.

Level 2: If-Else Statements

An if-else statement allows you to execute code based on a condition. Here is an example:

if (myNumber > 0) {
    System.out.println("Positive number");
} else {
    System.out.println("Negative number");
}

Try creating an if-else statement that checks if myVariable is greater than 5 and prints "Greater" if true, otherwise prints "Smaller or Equal".
